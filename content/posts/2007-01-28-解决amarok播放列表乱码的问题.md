FROM: <http://www.hiweed.com/node/111>

安装必要的软件

    % sudo apt-get install python-mutagen

把存在NTFS分区上的音乐文件软连接到ext2分区，

    % ln -s /media/win_e/music /sun/music
 
转换标签格式

    % cd /sun/music
    % find .  -iname  "*.mp3"  -execdir  mid3iconv  -e  GBK  {}  \;
